# Switch

按钮说明, 支持 Styling component

## 属性说明

| 属性     | 说明                                                                                   | 类型    | 默认值 |
| -------- | -------------------------------------------------------------------------------------- | ------- | ------ |
| loading  | loading 的样式 `['', 'bubbles', 'bars', 'spokes', 'spin']` 不写默认不显示 loading 图标 | ''      | false  |
| showLine | 是否显示线段,默认显示                                                                  | boolean | true   |
| children | 显示的内容                                                                             | node    | ''     |
