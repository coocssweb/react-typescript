# Image

Image, 图片支持

## 属性说明

| 属性    | 说明        | 类型                         | 默认值      |
| ------- | ----------- | ---------------------------- | ----------- |
| src     | 图片路径    | string                       | ``          |
| alt     | 说明文案    | string                       | ``          |
| display | 元素        | string`block`,`inline-block` | `block`     |
| width   | 宽度        | string/number                | `100%`      |
| height  | 高度        | string/number                | ``          |
| margin  | margin 间距 | array[string/number]         | [0, 'auto'] |
| onClick | 点击事件    | func                         | () => {}    |
