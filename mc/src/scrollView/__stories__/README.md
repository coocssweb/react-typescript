# Switch

按钮说明, 支持 Styling component

## 属性说明

| 属性      | 说明                       | 类型 | 默认值 |
| --------- | -------------------------- | ---- | ------ |
| threshold | 距离底部位置多少时触发加载 | number   | 50  |
| children  | 显示的内容                 | node | ''     |

## 方法说明

| 属性         | 说明                                          | 默认值        |
| ------------ | --------------------------------------------- | ------------- |
| onScroll     | onScroll 滚动条监听事件,放回坐标              | ({x,y}) => {} |
| onEndReached | onEndReached 加载数据，当到底部时会触发此方法 | ({x,y}) => {} |
