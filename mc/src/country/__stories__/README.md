# Country Index

城市索引组件

## 属性说明

| 属性      | 说明                             | 类型  | 默认值 |
| --------- | -------------------------------- | ----- | ------ |
| countries | 城市数组 如 [{name : 'Algeria'}] | array | []     |

## 方法

| 属性     | 说明                      | 类型                    | 默认值 |
| -------- | ------------------------- | ----------------------- | ------ |
| onChange | change 事件触发的回调函数 | (country: string): void | 无     |

### tips: 使用时一定要给外层容器高度
