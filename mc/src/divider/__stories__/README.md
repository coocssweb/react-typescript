# Divider

用于实现实现分割线

## 属性说明

| 属性       | 说明 | 类型                       | 默认值             |
| ---------- | ---- | -------------------------- | ------------------ |
| type       | 类型 | [`horizontal`, `vertical`] | `horizontal`       |
| color      | 颜色 | string                     | rgba(0, 0, 0, .12) |
| borderSize | 粗细 | number                     | 0.5px              |
| size       | 尺寸 | number                     | 0 默认撑开         |
| margin     | 间距 | [string/number]            | [0, 0]             |
