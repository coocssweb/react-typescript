# Svg

Svg 渐变

## Svg 属性说明

| 属性       | 说明                           | 类型          | 默认值 |
| ---------- | ------------------------------ | ------------- | ------ |
| src        | svg 路径                       | string        | ``     |
| width      | svg 宽度，会转为 rem           | number        | 200    |
| stopColors | 渐变色，只支持最多两种颜色渐变 | array[string] | []     |
