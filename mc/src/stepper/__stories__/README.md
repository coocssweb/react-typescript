# Stepper

`Stepper` 的其他组合组件:

- `Step`
- `StepLabel`
- `StepContent`

## Stepper

| 属性             | 说明                    | 类型                    | 默认值     |
| ---------------- | ----------------------- | ----------------------- | ---------- |
| activeStep       | 当前步骤                | number                  | 0          |
| alternativeLabel | 突显文本 （水平步骤条） | boolean                 | false      |
| direction        | [垂直间距, 水平间距]    | `horizontal` `vertical` | `vertical` |
