# Dialog box

## API

| 属性     | 说明                                | 类型   | 默认值 |
| -------- | ----------------------------------- | ------ | ------ |
| low      | 低电量模式                          | bool   | false  |
| size     | 大小                                | string | small  |
| quantity | 数值不带%号，例如 80% quantity={80} | number | 100    |
