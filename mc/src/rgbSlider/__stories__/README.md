# RgbSlider

考虑体验和性能问题，rgbSlider 采用节流的方式（即，在 500ms 内请求一次的形式。

## 属性说明

| 属性         | 说明     | 类型        | 默认值    |
| ------------ | -------- | ----------- | --------- |
| disabled     | 禁用状态 | boolean     | false     |
| defaultValue | 默认颜色 | string`hex` | `#ff0000` |

## 事件说明

| 属性     | 说明         | 类型             | 默认值   |
| -------- | ------------ | ---------------- | -------- |
| onChange | 修改事件回调 | func,(hex) => {} | () => {} |
