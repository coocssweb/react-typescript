# Slider

滑块

## 属性说明

| 属性         | 说明           | 类型    | 默认值 |
| ------------ | -------------- | ------- | ------ |
| min          | 最小值         | number  | 0      |
| max          | 最大值         | number  | 100    |
| defaultValue | 默认值         | number  | 0      |
| disabled     | 禁用           | boolean | false  |
| tooltip      | tooltip 提示框 | bool    | false  |

## 方法

| 名称     | 说明                | 默认值   |
| -------- | ------------------- | -------- |
| onChange | slider 值修改时触发 | () => {} |
