# 密码组件

| 属性   | 说明     | 类型   | 默认值 |
| ------ | -------- | ------ | ------ |
| length | 密码长度 | number | 6      |

## 方法

| 方法名     | 说明           | 类型 | 默认值   |
| ---------- | -------------- | ---- | -------- |
| onChange   | 密码框输入变化 | func | () => {} |
| onClear    | 清除密码框     | func | () => {} |
| onComplete | 密码框输入完成 | func | () => {} |
